# 🚕 方块出行

> 方块世界里的网约车司机模拟。

🎮 在线游玩：https://manji1233.github.io/Ride-hailing/voxel-world/

## 玩法

开黄色出租车沿椭圆环线公路，接送三位乘客回家。乘客在候车点等候，送到房子外的路边即可，每单记入可审计的证据链。

- **自由模式**：无压力，随意接送、探索风景。
- **限时模式**：每单递减 60→45→30 秒，超时乘客抱怨。

## 世界

体素开放世界：中央椭圆湖+石拱桥，三条分支路通向海边沙滩、乡村农场、小镇街市。牛羊漫游、小动物横穿、昼夜切换，世界是活的。

## 自动驾驶

按 T 一键开启：自动走最短路线接送，**仅在公路上行驶**，拟人驾驶（看远转弯、弯前减速、弧线掉头），过桥沿桥面直行。

## 视角

按 V 切换跟车尾 / 车内方向盘 / 俯视。车内可见仪表盘、方向盘、双手随转向转动，低头能看见自己的身体。

## 操作

**WASD** 移动 · **F** 上下车 · **V** 切视角 · **T** 自动驾驶 · **N** 昼夜 · **R** 重置
