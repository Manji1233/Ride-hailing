# 方块出行 · 体素网约车世界

> 一个用方块拼出来的开放世界网约车模拟游戏。你是一名司机，开着黄色出租车沿环线公路接送三位乘客，穿越海边、乡村、小镇三种风景。

🎮 **在线游玩**：https://manji1233.github.io/Ride-hailing/voxel-world/

---

## 一、构思与设计

### 世界观
整个世界是一张 **432×432 的体素地图**，中央一座不规则椭圆湖，湖水之上架着一座**中式石拱桥**作为车行桥，连接环线两侧。一条**扁椭圆环形公路**绕场一周，三条分支路向外延伸到三个主题目的地：

| 分支 | 主题 | 风景特色 |
|------|------|----------|
| 海边 | 夏日沙滩别墅 | 沙滩、椰子树、木栈桥、躺椅、排球网、沙堡 |
| 乡村 | 农场木屋 | 牛羊马、拖拉机、干草垛、稻草人、水井、风车 |
| 小镇 | 乡村小镇 | 钟楼、喷泉广场、集市摊位、果树、街灯、民居 |

### 视觉风格
- **体素（Voxel）美术**：所有地形、建筑、车辆、人物、动物都用 1×1×1 方块堆叠而成，InstancedMesh 高性能渲染。
- **治愈系配色**：明亮的草地绿、湖水蓝、出租车黄，昼夜两套光照与雾化。
- **细节人物**：司机有完整身体（制服、司机帽、双手搭方向盘、双腿双脚），车内视角可见自己的身体；乘客有背包、斜挎带、五官。

### 车内沉浸
按 **V** 切换到**车内方向盘视角**，可见拟真车内设施：
- 仪表台 + 车速表盘 + 随车速转动的红色指针
- 中控屏、出风口、档把、中控台
- 双前座椅（带头枕）+ 后排长椅
- 左右门板 + 把手 + 车内后视镜
- 方向盘（轮缘+轮毂+辐条），转向时**双手和前臂随方向盘一起转动**
- 内饰相对相机固定，只有车外风景流动、方向盘和手臂随转向变动——真实驾驶视角

## 二、核心玩法

### 目标
从网约车场站出发，沿环线依次找到三位候车乘客，接他们上车，送到各自的家门口（房子外的路边下车点），赚取金币。

### 两种模式

**🟢 自由模式**
- 无时间压力，随意接送、自由探索四段风景。
- 适合观光、熟悉地图。

**🟡 限时模式**
- 每位乘客有送达时限，**按实际送达顺序递减**（顺序不固定）：
  - 第 1 单：**60 秒**
  - 第 2 单：**45 秒**
  - 第 3 单：**30 秒**
- 超时乘客会抱怨，满意度下降；按时送达 +10 金币奖励。

### 自动驾驶（按 T 开关）
一键开启拟人自动驾驶，车会自动：
- **以最短路线接送**：每次自动接最近的候车乘客，顺序不固定。
- **仅在公路上行驶**：硬约束在路面内，偏离即拉回，绝不驶离公路。
- **拟人驾驶**：纯追踪前瞻转向（看远不看近）、方向盘渐进平滑、弯道预判减速、掉头走小半径弧线。
- **过桥直行**：上桥后沿桥面中线直行到对岸桥头，再转弯接回环线。
- 全部送达后自动过桥去对岸继续。

### 治理与证据链（网约车主题）
- 每次接单/送达生成可审计的**证据链**（哈希链）。
- 可运行审计、校验证据链、模拟篡改、发起申诉。
- 实时治理指标：金币、完成单数、压单率、公平指数、满意度、抱怨数。

## 四、世界细节

### 道路系统
- **扁椭圆环线**：A/B 半轴比约 2.2，明显偏扁，转弯更自然。
- **加宽路面**：9 格宽，会车宽松。
- **中央车行桥**：石拱桥跨湖，桥面平整可通行，桥头石柱栏杆。
- **分支路**：从环线三个路口向外延伸到主题目的地。

### 乘客系统
- 三位乘客在环线**候车点**等候（刻意远离各分支路口，不在目标路口附近）。
- 目的地为**房子外的路边下车点**——送到路边即可，不必开到房子门口；小镇下车点在镇外路口。
- 小地图实时显示候车乘客（红）、目的地（绿）、出租车（黄三角）。

### 生命的世界
- **NPC 与动物自由漫游**：小镇居民、农场牛羊马、钓鱼人等在各自范围内自由走动，腿部摆动。
- **路面动物**：6 只动物在路肩漫游，偶尔会挡路（每局最多 2 次），3 秒后离开。
- **横穿小动物**：随机事件，小鹿/狐狸/野猫/野狗横穿路面，靠近会停顿，撞到扣金币。
- **昼夜系统**：白天/夜晚两套光照，灯光招牌夜晚发光。
- **云朵漂移、船只、风车转动**等动态环境。

### 导航地图（左上）
- 俯视示意图，显示环线、分支路、三个主题目的地。
- 目的地用**语义图标**：海边（太阳+海浪）、乡村（谷仓+麦穗）、小镇（两栋房屋），下方文字标签。
- 当前送达目的地有**脉冲高亮环**。
- 图例：出租车（放大显示）、候车乘客、目的地。

## 五、技术亮点

- **纯前端**：Three.js + 原生 JS，单文件 game.js，无需后端。
- **高性能体素**：InstancedMesh 合并渲染数十万方块，DynamicDrawUsage 动态更新。
- **Perlin 噪声地形**：自然起伏的山地、湖泊、植被分布。
- **拟人自动驾驶**：纯追踪算法 + 弯道曲率预判 + 公路硬约束，转向和加减速连续平滑。
- **本地运行**：双击 `打开世界.bat` 启动本地 HTTP 服务，或直接访问在线网址。

## 六、本地运行

```bash
# 方式一：双击启动
打开世界.bat   # 启动本地服务器并打开浏览器

# 方式二：手动
python -m http.server 8765
# 浏览器访问 http://localhost:8765/voxel-world/
```

> ⚠️ 请通过本地服务器（http://）访问，不要直接双击 index.html（file:// 协议下 ES Module 会被浏览器拦截）。
