# 🚕 方块出行

> **当一名方块世界的网约车司机，把三位乘客安全送回家。**
> 沿着扁扁的椭圆环线公路，穿越海边、乡村、小镇三种风景，赚你的方块金币。

🎮 **立刻开玩** → https://manji1233.github.io/Ride-hailing/voxel-world/

---

## 🌍 这是个什么世界？

一张用方块拼出来的开放世界。中央一汪不规则椭圆湖，湖上一座中式石拱桥跨水而过。一条**扁扁的椭圆公路**绕场一周，三条岔路各自伸向远方——

- 🏖️ **海边**：金色沙滩、椰子树、伸进海里的木栈桥、排球网、沙堡，还有个戴草帽的钓鱼人。
- 🌾 **乡村**：低头吃草的牛羊马、红色拖拉机、金黄干草垛、稻草人、吱呀转动的风车。
- 🏘️ **小镇**：钟楼、喷泉广场、彩色顶棚的集市摊位、果树下打盹的野猫野狗、暖黄街灯。

而你，开着一辆亮黄色出租车，在这方块天地里接单、送客、看风景。

---

## 🎯 你要干什么？

简单：**找到三位候车的乘客，接上车，送到他们家。**

但细节里全是戏——

- 乘客站在环线上的候车点，**故意离路口远远的**，你得开过去接。
- 送到**房子外边的路边**就行，不用顶到人家门口（小镇那位尤其矜持，下车点在镇外路口）。
- 每一单都会记进一条**可审计的证据链**，谁接的、送哪了、用了多久——公平派单，全程留痕。

### 两种活法

| 模式 | 适合谁 | 说明 |
|------|--------|------|
| 🟢 **自由模式** | 悠闲党 | 没时间压力，慢慢开、到处逛，接不接单随你。 |
| 🟡 **限时模式** | 挑战党 | 每单有时限，**越送越紧**：第 1 单 60 秒 → 第 2 单 45 秒 → 第 3 单 30 秒。超时乘客会吐槽你。 |

顺序不固定——你先送谁，谁就是"第 1 单"。

---

## 🤖 不想自己开？让 AI 替你开

按 **T** 键，自动驾驶接管。这辆车会自己：

- 🧭 **挑最近的路**：每次自动接离你最近的乘客，环线上前进还是后退，哪个短走哪个。
- 🛣️ **死也不出公路**：硬约束在路面里，偏一点就被拉回来，**绝不会冲进草地撞树**。
- 🧠 **开得像个人**：看远处转弯（纯追踪）、方向盘平滑打、进弯前减速、掉头走弧线不原地转圈。
- 🌉 **过桥不耍花活**：上桥沿桥面中线笔直开到对岸，到桥头才拐弯接回环线。

你可以切到车内视角，看 AI 怎么握着方向盘过弯——挺解压的。

## 🎨 钻进车里看看

按 **V** 切到**车内方向盘视角**，你会看到：

- 仪表台上**车速指针随速度实时转动** 🔴
- 中控屏亮着蓝光、出风口、档把、中控台
- 双前座椅带头枕、后排长椅、左右门板把手、车内后视镜
- 最关键——**你的司机有身体**：制服、黄色司机帽、双眼，双手搭在方向盘上，**转弯时双手和前臂跟着方向盘一起转**，低头还能看见自己的腿和脚。

内饰是钉死的，只有车外风景在往后流、方向盘在你手里转——就像真坐在驾驶座上。

## 🗺️ 左上角的小地图

一张俯视图，帮你认路：

- 黄色三角是你（出租车，带朝向）。
- 红点是候车乘客，绿点是目的地。
- 三个目的地有专属小图标：**海边**（太阳+海浪）、**乡村**（谷仓+麦穗）、**小镇**（两栋小屋），底下还标着名字。
- 正在送的那位，目的地会**一闪一闪**高亮。

## 🦙 这个世界是活的

- 牛羊马在农场溜达，小镇居民街头闲逛，腿部一摆一摆地走。
- 路边 6 只动物自由漫游，偶尔会**蹿到路中间挡你道**（一局最多两次，3 秒后挪窝）。
- 山林里随时可能窜出小鹿、狐狸、野猫野狗**横穿马路**，靠近会愣一下，撞上了扣金币。
- **昼夜切换**（N 键）：白天明亮治愈，夜晚灯牌亮起、雾气沉沉，另有一番味道。
- 云在天上飘，船在湖面荡，风车吱呀转。

## 🎮 怎么玩

| 按键 | 干嘛 |
|------|------|
| **W / S** | 前进 / 后退（长按 W 3 秒飙车） |
| **A / D** | 转向 |
| **空格** | 手刹 / 跳跃 |
| **F** | 上车 / 下车 |
| **V** | 切视角：跟车尾 → 车内 → 俯视 |
| **T** | 自动驾驶 开 / 关 |
| **N** | 昼夜切换 |
| **R** | 重置世界 |
| **左键/右键** | 步行时拆/放方块 |
| **ESC** | 释放鼠标 |

**新手流程**：进游戏选模式 → 走近黄色出租车按 F 上车 → 看小地图找红点乘客 → 开过去接客 → 沿绿点送到家 → 重复三次 → 全部送完，AI 自动过桥去对岸。

## ✨ 背后的小心思

- 纯前端，Three.js + 一个 game.js 文件，不用装任何东西。
- 几十万方块用 InstancedMesh 一次画完，浏览器带得动。
- 自动驾驶是手写的拟人算法，不是接 API——纯逻辑让它"开得像人"。
- 地形靠 Perlin 噪声生成，每座山每棵树都不太一样。
