方块出行

场站出发，沿环线依次经过 城郊市井 → 山谷森林 → 滨湖度假 → 返程田野 四段差异化风景。靠近黄色出租车按 F 上车，开到候车点接客、送到其家赚金币。可自由折返，每一次派单生成可审计的证据链。

  • WASD 移动 / 驾驶（长按 W 3 秒加速）
  • 空格 跳跃 / 手刹
  • 鼠标 视角
  • 左键 拆方块
  • 右键 放方块
  • F 上/下车
  • V 切换跟车 / 车内方向盘 / 俯视
  • T 自动驾驶
  • N 昼夜
  • R 重置
  • ESC 释放鼠标
